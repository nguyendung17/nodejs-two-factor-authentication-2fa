# NodeJS Two-Factor Authentication (2FA)

## Requirements

* **Nodejs > v12.0.x**

Clone project and run test.

```
$ git clone https://git_url_clone <project_dir>
$ cd <project_dir>
$ npm install
$ node src/server.js
```
